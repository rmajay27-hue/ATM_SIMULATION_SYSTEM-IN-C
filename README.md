# ATM Simulation System using C

## Description
This mini project simulates basic ATM operations such as checking balance,
depositing money, and withdrawing money using C language.

## Features
- Menu-driven program
- Check account balance
- Deposit and withdraw money

## Concepts Used
- Functions
- Switch case
- While loop
- Conditional statements

## How to Run
gcc atm.c -o atm
./atm
